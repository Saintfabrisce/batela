# Node.js Starter Project

A simple Node.js starter project with Express.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. For production:
   ```bash
   npm start
   ```

## Features

- Express server
- CORS enabled
- Environment variables support
- Jest for testing
- Nodemon for development
